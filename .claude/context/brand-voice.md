# Voix éditoriale & contenu

## Principe central
**Contenu unique par site ET par langue.** Les faits (prix, specs, légalité) sont des variables partagées ; la prose éditoriale est générée et différenciée par site (angle, ton, structure) pour que chaque site paraisse distinct (pas un réseau cloné).

## Par site
Chaque site a une identité : persona, ton, vocabulaire, structure de page privilégiée (un site met les avis en avant, un autre un guide d'usage, un autre un comparatif). Définir l'angle de chaque NDD quand Florian les fournit.

## Multilingue (FR/EN/ES)
Traductions natives, pas littérales. Adapter le registre par langue/marché.

## Variation lexicale (anti-duplication)
Faire tourner via dictionnaires par langue :
- Verbes : acheter / commander / trouver / obtenir / se procurer / où acheter
- Noms : weed / cannabis / marijuana / hash / fleur / herbe / beuh
- Promo : code promo / code réduction / coupon / bon de réduction / rabais / deal

## Règles dures
- **YMYL / santé** : pages usages (sommeil, douleur…) avec disclaimers, ton prudent, **aucune allégation médicale**, sources fiables.
- **E-E-A-T** : pages légales exactes (cadre fédéral + règles par province), à jour, crédibles.
- **CTA** : clairs, orientés conversion vers les shops en ligne avec livraison (liens affiliés), sans surpromesse.
- Persona projet : « Michelle Lanoy » (contenu/reviews SEO sur les shops canadiens).
