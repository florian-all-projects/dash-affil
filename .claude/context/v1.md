# Périmètre V1 (MVP)

## But du V1
Prouver la chaîne de bout en bout sur **1 site pilote** avant d'industrialiser les 9 autres.

## Inclus dans le V1
1. **Fondations** : serveur + PostgreSQL + schéma + orchestration.
2. **Scraper** : cascade (Shopify/Woo/JSON-LD) sur les shops 🟢 ACTIFS uniquement au départ, + dédup + détection de changements.
3. **Génération** : contenu (faits/éditorial, FR au minimum puis EN/ES) + 2 images/produit.
4. **1 site Astro complet** : catalogue produits + pages villes + pages coupons, avec son design propre.
5. **SEO de base** : sitemap, schema, hreflang, maillage interne.
6. **Déploiement** + cron hebdo + rapport de run.

## Critères de réussite V1
- Catalogue complet d'au moins les shops actifs, sans doublon, boutons affiliés fonctionnels (deep-link + ref).
- Prix qui se mettent à jour automatiquement au scrap hebdo.
- Pages villes et coupons générées, contenu unique, indexables.
- Site rapide (CWV verts), structuré pour le SEO.

## Hors V1 (plus tard)
- Les 9 autres sites (industrialisation, Phase 4 du PLAN).
- Axes additionnels avancés (comparateur de prix multi-sites élaboré, strains exhaustifs) au-delà du socle.
- Optimisations GEO/IA poussées.

## Dépendances
NDD + angle du site pilote, clés API, données shops physiques (pour les pages villes), accès serveur. Voir `progress.md`.
