# Stratégie SEO

## Objectif
**Ranker au-dessus des shops affiliés eux-mêmes** sur les requêtes commerciales et locales.

## Leviers
- **Programmatique** : pages villes (« acheter weed + [ville] », tout le Canada), fiches variétés/strains, pages coupons (« code promo [shop] »), comparateur de prix.
- **Maillage interne en silos** : page ville ↔ fiches produits ↔ catégories ↔ villes voisines ↔ hub provincial ↔ accueil.
- **Contenu riche & unique** par site/langue (voir brand-voice) ; éviter pages minces/doorway (richesse réelle + données locales + variation lexicale).
- **Fraîcheur** : mise à jour hebdo (prix, nouveautés, coupons vérifiés).
- **E-E-A-T** : pages légales exactes, avis, guides d'achat.

## Technique
- Données structurées : `Product`, `Offer`, `LocalBusiness`, `FAQPage`, `BreadcrumbList`.
- `hreflang` FR/EN/ES, URLs par langue, sitemaps, robots propres.
- Sites statiques rapides (Astro).

## GEO / IA
Optimiser aussi pour AI Overviews / ChatGPT / Perplexity (réponses structurées, entités claires).

## Outils dispo (MCP)
- **Ahrefs** (site explorer, keywords, rank tracker, brand radar, web analytics).
- **Haloscan** (keywords, domaines, SERP, GMB backlinks).
Utiliser pour la recherche de mots-clés, le suivi de positions et l'analyse concurrentielle (shops affiliés).
